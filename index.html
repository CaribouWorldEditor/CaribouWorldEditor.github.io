Caribou

World Editor

Caribou World Editoris a world editor designed to makemapsfor theGodot Game EngineusingConstructive Solid Geometrytechnices inspired by other world editors for other different engines. It is built using Godot and it provides many simple and advanced tools to create geometry. The maps created in theCaribou World Editorare directly exported asgodot scenesin order for the use to not be required to install any additional packages or addons.

Made by Hristo Katsarski and Zlatan Dimitrov.
