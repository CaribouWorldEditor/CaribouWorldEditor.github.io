Caribou

World Editor

Caribou World Editor is a world editor designed to make maps for the Godot Game Engine using Constructive Solid Geometry technices inspired by other world editors for other different engines. It is built using Godot and it provides many simple and advanced tools to create geometry. The maps created in the Caribou World Editor are directly exported as godot scenes in order for the use to not be required to install any additional packages or addons.
